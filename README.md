# Kavya Kanaja (ಕಾವ್ಯ ಕಣಜ) - National Pride 🌿

**Kavya Kanaja** (Poetry Granary) is a "Literary Revival" Android application designed to bridge the gap between Karnataka's rich literary heritage and the digital-native generation. Developed as part of the **MindMatrix VTU Internship Program**, this app transforms classical Kannada poetry into an engaging, multimodal experience.

## 🚀 Vision
Karnataka possesses a 2,000-year-old literary history, yet "Old Kannada" (Halegannada) is becoming increasingly difficult for Gen-Z to access. Kavya Kanaja makes poetry "cool" again by providing a distraction-free, aesthetically pleasing environment for linguistic exploration.

## ✨ Key Features
- **Poem of the Day:** A curated daily verse to encourage consistent engagement with literature.
- **Listen & Learn:** Integrated audio recitations (via Media3 ExoPlayer) to help users master the rhythm and pronunciation of Kannada verses.
- **Bhavartha (Explanation):** Concise philosophical summaries that decode deep meanings for modern readers.
- **Interactive Vocabulary:** Tap-to-reveal meanings for complex and archaic Kannada words.
- **Poet’s Corner:** Detailed biographies of Jnanpith awardees and legendary Rashtrakavis like Kuvempu and Bendre.
- **100% Offline:** A local JSON-based architecture ensures the "Granary" is accessible anywhere, even without internet.

## 🛠️ Technologies Used
- **Language:** Kotlin
- **UI Framework:** Jetpack Compose (Material 3)
- **Architecture:** MVVM (Model-View-ViewModel)
- **Data Management:** Local JSON Database with GSON parsing
- **Audio Engine:** Media3 ExoPlayer
- **Design System:** custom book-inspired color palette & typography
- **Development Tool:** Android Studio (Ladybug/Koala)

## 📥 Installation
1. Download the latest APK from the [Releases](https://github.com/likki-dotcom/Kavya-Kanaja/releases) section.
2. Enable "Install from Unknown Sources" on your Android device.
3. Open the APK and enjoy the world of Kannada poetry!

## 🧑‍💻 For Developers
To run this project locally:
1. Clone the repository:
   ```bash
   git clone [https://github.com/likki-dotcom/Kavya-Kanaja.git](https://github.com/likki-dotcom/Kavya-Kanaja.git)




   how to run
   1. Prerequisites
Android Studio: Download the latest version (Ladybug or Koala recommended).

JDK: Java Development Kit 17 or higher.

SDK: Android SDK with API Level 34 (Upside Down Cake) or higher.

2. Clone the Repository
Open your terminal or command prompt and run the following command:

Bash
git clone https://github.com/likki-dotcom/Kavya-Kanaja.git
3. Import the Project
Open Android Studio.

Select File > Open and navigate to the cloned Kavya-Kanaja folder.

Wait for the Gradle Sync to finish. This might take a few minutes as it downloads all the necessary libraries (GSON, Media3, etc.).

4. Run on Device/Emulator
Connect an Android device via USB (ensure USB Debugging is enabled) or start a Virtual Device (AVD).

Click the Run button (Green Play Icon) in the top toolbar.

The app will compile, install, and launch the "Poem of the Day" screen automatically.
